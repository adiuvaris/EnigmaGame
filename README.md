## Description

This puzzle was invented by Douglas A. Engel and it consists of two intersecting disks in a plastic holder.
This program is a computer version of this puzzle that I wrote in Python.


## Install

Via PyPi

``` bash
$ pip install EnigmaGame
```

## Usage

``` python
python -m EnigmaGame
```
